# js_rpc_drive
js逆向通杀免补环境工具(免去抠代码补环境)    
支持高并发 兼容性较强    
环境安装:    
pip install -r requirements.txt    
config.json 配置详解:   
  ON_OFF 是否无头   
  USER_DATA_PATH 浏览器临时缓存地址(可以不动)   
  BROWSER_PATH 浏览器安装路径    
  URL  网站地址   
  COOKIES  网站cookies(不需要填{}即可)         
扣的js文件放入jsdata文件 运行rpc_drive 即可     
调用详情见 dome文件
